# Premium Medical Clinic Website (NYC)

A high-performance WordPress custom theme for a cosmetic procedure clinic located in New York.

## 🚀 Technologies
- **PHP / WordPress** - Custom theme logic
- **Tailwind CSS** - Utility-first styling
- **JavaScript & GSAP** - High-end interactive animations
- **Schema.org (JSON-LD)** - Advanced SEO integration

## 🛠 Features
- **Core Web Vitals Optimized:** Fast loading speeds and zero layout shifts.
- **GSAP Animations:** Smooth scroll interactions and element transitions.
- **Custom Post Types:** Easy management for clinic services and doctors.
- **Local SEO:** Fully configured Schema for better visibility in NYC search results.

## 📈 Performance
The project focuses on high Google Lighthouse scores, specifically targeting LCP and CLS metrics.
